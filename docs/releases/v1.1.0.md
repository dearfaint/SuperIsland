# SuperIsland v1.1.0

这是面向 Apple Silicon Mac 的功能预览版本，安装包为本地 ad-hoc 签名，尚未经过 Apple Developer ID 公证。首次打开如果被 macOS 拦截，请在 Finder 中右键选择“打开”，或到“系统设置 -> 隐私与安全性”允许打开。

## 下载

- `SuperIsland-1.1.0-arm64.dmg`
- SHA-256: `3860d8933fad76a7d1268f2a40dcc32919c257620b18f816d2e9dcab784a49e0`

## 主要新增

- 默认 README 改为简体中文，并保留英文说明到 `README.en.md`。
- 天气模块新增“自动选择 / 和风天气 / Open-Meteo / 彩云天气”数据源设置。
- 和风天气改用 JWT 凭据，支持实时天气、逐小时预报、每日高低温、空气质量和气象预警。
- 中国大陆天气可优先使用和风天气，Open-Meteo 作为免费默认源和备用源。
- 天气弹窗和主页天气增加预警图标，鼠标悬停可查看预警详情。
- 新增本地第三方扩展安装流程，支持权限预览、安装后立即启用、卸载扩展文件和持久化数据。
- 新增可手动安装的 Nudge / 稍后提醒扩展，支持快捷延时、自定义分钟数、完成、延后十分钟和提示音设置。
- 新增 Computer Status 扩展，显示 CPU、内存、磁盘、SoC 温度、风扇转速、电源、运行时间和热状态。
- 新增 HK + A Stocks 扩展，支持港股、沪市、深市代码，优先使用非大陆行情接口。

## 修复与优化

- 修复紧凑模式横向切换和模块停止后的状态冲突。
- 修复扩展启停状态在菜单栏和设置页之间不一致的问题。
- 优化股票紧凑视图，保持原始宽度并显示市场前缀代码。
- 优化电脑状态刷新时的数值跳动、阴影闪烁和布局稳定性。
- 修复 Shelf 对文本、源代码、JSON 等内容类型文件的拖放暂存。
- 保留中文输入法组合文本，避免扩展视图刷新时打断正在输入的拼音。
- 修复本地 arm64 DMG 打包流程，确保 App 和内置 Node 运行时架构一致，并在无证书环境下完成临时签名。

## 已知限制

- 当前 DMG 仅包含 Apple Silicon arm64 构建。
- 当前 DMG 未经过 Apple Developer ID 公证，不适合作为正式签名发布包。
- Intel Mac 需要自行使用 `scripts/build-dmg-intel.sh` 构建。
